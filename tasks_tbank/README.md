# TBank AQA training

Алгоритмические задачи и тренировка под экзамены/контесты T‑Bank.

- Язык: Python 3.13
- Формат: stdin → `sys.stdin.read()`, stdout → `print`
- Цель: научиться быстро доводить идею до рабочего кода и понимать сложность решений.

Структура:
- `tasks_tbank/` — решения задач и тестовые STDIN-файлы.
